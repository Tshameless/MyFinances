# Python 量化版本路线图

## 当前状态

项目已经从最初的实验性原型收敛为一个可本地运行的日频回测工具，当前版本重点解决以下问题：

1. 真实 CSV 数据接入
2. 交易日对齐和数据校验
3. 多因子评分与等权持仓
4. 固定周期调仓与交易成本处理
5. 净值和调仓日志输出
6. 基础回归测试

## 当前边界

当前版本适合做研究和本地验证，还没有覆盖以下生产级能力：

1. 实时行情抓取
2. 券商下单接口
3. 复杂仓位约束
4. 数据库存储
5. 图形化报表
6. 组合级风险归因

## 推荐下一阶段

### 阶段 1：数据增强

1. 增加复权价格支持
2. 区分停牌、涨跌停和不可交易
3. 接入 A 股、ETF、加密等不同市场的数据适配层

### 阶段 2：策略增强

1. 支持更多因子
2. 支持行业中性和风险暴露约束
3. 支持自定义打分和权重模型

### 阶段 3：回测增强

1. 引入基准比较
2. 引入更多绩效指标
3. 支持分层成本和税费模型

### 阶段 4：工程化

1. 增加 CI
2. 增加 lint 和类型检查
3. 增加样例数据和更完整文档
