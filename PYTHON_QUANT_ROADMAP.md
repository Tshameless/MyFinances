# Python 量化版本路线图

## 当前状态

项目已经从实验性原型收敛为一个可本地运行的 A 股日频多因子回测与报告工具。当前版本已经覆盖：

1. 本地 CSV 行情、基准、动态股票池和代码分组映射接入
2. 日期解析、重复行、价格字段、交易日对齐、执行价覆盖率和映射质量校验
3. 收盘价/复权价估值，以及收盘价、复权价、开盘价、VWAP 成交价建模
4. 信号日和成交日分离，支持延迟若干个对齐交易 bar 后执行
5. 动量、均值回归、低波动三因子评分，外部评分 CSV，因子权重覆盖和高分/低分选股方向对照
6. Top N 等权目标、目标现金权重、单票目标权重上限和单组最多入选数量
7. 现金、股数、整手、T+1、停牌、可买可卖、涨跌停和成交量参与率约束
8. 佣金、买卖不同佣金、最低佣金、滑点、市场冲击、过户费和卖出印花税
9. 净值、调仓、持仓、交易、未成交、绩效、运行清单、SVG 图表和 HTML 报告输出
10. 回撤、月度收益、滚动风险、相对基准、执行质量、换手、持仓周期、暴露、归因、成本和盈亏对账分析
11. 因子 IC、Rank IC、分组收益、衰减和相关性分析
12. 策略健康诊断、风险闸门、参数 sweep、排行榜、热力图、walk-forward 验证和 walk-forward 优化
13. `unittest` 回归测试、Ruff/Mypy 配置和开发检查脚本

## 当前边界

当前版本适合本地研究、参数试验和报告归档，还没有覆盖以下生产级能力：

1. 行情源自动抓取、数据库存储、增量更新和数据版本管理
2. 复权因子、分红拆股、上市/退市生命周期、指数成分历史和公司行动事件管理
3. 策略插件接口、自定义组合构建器、组合优化器和风险模型
4. 行业/风格中性化、暴露目标、风险预算、换手约束和目标波动率控制
5. 分钟级、逐笔级、订单簿、排队成交、部分成交和完整订单状态机
6. 券商下单、实盘风控、任务调度、权限、审计和监控告警
7. 面向交互分析的前端工作台、实验对比、参数管理和报告检索

## 推荐下一阶段

### 阶段 1：策略扩展接口

1. 抽象策略协议，让选股、打分、过滤、目标权重生成可以由插件实现
2. 把当前三因子 Top N 策略保留为默认内置策略
3. 支持多列自定义因子、标准化流水线和策略级参数 schema
4. 为策略接口增加端到端样例和回归测试

### 阶段 2：数据增强

1. 引入标准数据目录、数据字典和样例数据集
2. 增加行情数据版本、增量更新和输入文件哈希索引
3. 支持上市、退市、指数成分变更和复权因子历史
4. 预留 A 股、ETF、可转债、港股或加密等市场适配层

### 阶段 3：组合与风险增强

1. 支持行业中性、风格暴露约束和组合级风险预算
2. 增加目标波动率、最大换手、最小持仓期和交易惩罚项
3. 支持基于协方差矩阵的最小方差、风险平价或最大夏普组合
4. 在策略健康诊断里区分信号风险、组合风险、执行风险和数据风险

### 阶段 4：回测真实度增强

1. 引入订单状态机，区分目标、委托、成交、撤单和拒单
2. 支持部分成交、日内成交窗口、涨跌停排队和更细粒度容量模型
3. 增加分钟级或事件驱动回测适配层
4. 增加成本模型校准报告和成交假设敏感性分析

### 阶段 5：工程化与体验

1. 拆分报告层，把 CSV、JSON、SVG、HTML 和解释文本继续分层
2. 增加 CLI、异常路径、报告字段、manifest 兼容性和端到端样例测试
3. 建立 CI、发布版本、变更日志和示例数据快照
4. 构建交互式实验工作台，用于浏览报告、比较参数、追踪实验和筛选风险闸门结果
